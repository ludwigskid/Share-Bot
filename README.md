## Installation


### Ubuntu
```bash
sudo apt update
sudo apt install golang-go
git clone https://github.com/ludwigskid/Share-Bot
cd Tiktok-Sharebot
go run .
```


### Windows
Go to the [Go download page](https://go.dev/dl/)
```bash
git clone https://github.com/ludwigskid/Share-Bot
go run .
```
